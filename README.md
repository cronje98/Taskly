# Taskly
Taskly is a simple web-based task manager built using Flask. It lets users add, view, and delete tasks(stored in a database) in a clean interface â€” perfect for learning and productivity.

# Features
- Automatically creates database 
- Add and delete tasks
- Responsive UI with HTML/CSS
- Built using Flask (Python backend)

# Tech Stack
- **Backend:** Python, Flask
- **Frontend:** HTML, CSS
- **Database:** SQLite
